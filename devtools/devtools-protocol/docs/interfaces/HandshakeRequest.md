# Interface: HandshakeRequest

Defined in: [handshake.ts:37](https://github.com/yoltra/yoltra/blob/5ed5f4e4cc19d06832097c4012b16d8a869f58c3/devtools/devtools-protocol/src/handshake.ts#L37)

Sent by a client immediately after WebSocket connection.

## Remarks

The hub validates the protocol version and responds with
a [HandshakeResponse](HandshakeResponse.md). Connections that do not handshake
within 5 seconds are terminated.

Exactly one of `store` or `extension` must be present depending
on the declared [DevtoolsRole](../enumerations/DevtoolsRole.md).

## Example

```ts
import type { HandshakeRequest } from "@yoltra/devtools-protocol";

const req: HandshakeRequest = {
  type: "HANDSHAKE_REQUEST",
  protocolVersion: "1.0.0",
  role: DevtoolsRole.STORE,
  store: {
    id: crypto.randomUUID(),
    name: "myStore",
    capabilities: { replay: true, stateSnapshot: true, emit: false },
  },
};
```
